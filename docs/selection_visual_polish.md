# Selection Visual Polish

## Objetivo

Pulir el feedback visual del cuerpo seleccionado.

## Cambios

- El halo seleccionado es menos invasivo en cuerpos grandes.
- Se mantiene visible en cuerpos pequeños.
- La opacidad del material se reduce para no tapar tanto el planeta.
- Se agrega un pulso sutil de escala.
- El pulso usa tiempo de simulacion y se mantiene dentro de un rango pequeno.

## Comportamiento esperado

- Sol, Jupiter y Saturno no quedan cubiertos por un halo excesivo.
- Lunas y cuerpos pequenos siguen teniendo indicador visible.
- El selector conserva lectura clara sin dominar la escena.
