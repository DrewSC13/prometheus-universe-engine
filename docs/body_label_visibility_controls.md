# Body Label Visibility Controls

## Objetivo

Reducir la saturación visual de etiquetas cuando se renderiza el Sistema Solar completo.

## Control

- L: alterna el modo de etiquetas.

## Modos

- major: muestra Sol y planetas principales.
- all: muestra todos los cuerpos, incluida la Luna.
- none: oculta todas las etiquetas.

## Estado

Implementado en SolarSystemRenderPlugin mediante LabelVisibilityMode.
