# Orbit Visibility Controls

## Objetivo

Controlar la saturación visual de órbitas durante la Fase 1.

## Control

- O: alterna el modo de órbitas.

## Modos

- all: muestra todas las órbitas.
- planetary: muestra solo órbitas alrededor del Sol.
- none: oculta todas las órbitas.

## Estado

Implementado en SolarSystemRenderPlugin mediante OrbitVisibilityMode.
