# Saturn Ring Markers

## Objetivo

Agregar una primera representación visual de anillos planetarios durante la Fase 1.

## Implementación

Saturno recibe dos anillos procedurales construidos con marcadores esféricos pequeños.

## Decisión técnica

Se usa una solución procedural simple antes de introducir texturas reales, shaders o meshes complejos.

## Ventajas

- No requiere assets externos.
- Mantiene el render liviano.
- Es fácil de probar.
- Se integra con el sistema actual de cuerpos visuales.

## Próximos pasos

- Agregar textura de anillos.
- Añadir inclinación axial realista.
- Añadir anillos de Urano.
- Añadir lunas principales.
