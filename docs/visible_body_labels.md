# Visible Body Labels

## Objetivo

Hacer que las etiquetas del Sistema Solar sean más legibles durante la Fase 1.

## Cambios

- Etiquetas más grandes.
- Tamaño según clase de cuerpo.
- Color según clase de cuerpo.
- Mayor separación vertical.
- Orientación de etiquetas alineada con la cámara.

## Clases visuales

- Estrella: etiqueta más grande y amarilla.
- Planetas terrestres: etiqueta clara.
- Gigantes gaseosos: etiqueta cálida.
- Gigantes helados: etiqueta celeste.
- Satélites naturales: etiqueta gris.

## Control relacionado

- L: alterna etiquetas entre major, all y none.
