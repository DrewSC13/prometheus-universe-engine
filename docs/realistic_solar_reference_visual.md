# Realistic Solar Reference Visual

## Objetivo

Ajustar el Sol para que se parezca más a una esfera incandescente realista:

- superficie granular de plasma,
- manchas solares oscuras,
- borde brillante amarillo/naranja,
- halo/corona suave,
- menos nube de partículas alrededor.

## Decisiones visuales

- La superficie usa más celdas pequeñas para parecer textura continua.
- El tercer material solar representa manchas oscuras.
- La corona se compacta cerca del borde para evitar efecto de “bolas flotantes”.
- El halo por capas sostiene el brillo exterior.
- La escala orbital realista comprimida no se modifica.

## Estado

Este corte se enfoca solamente en el aspecto del Sol y en reparar el enfoque contextual de lunas.
