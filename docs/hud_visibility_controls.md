# HUD Visibility Controls

## Objetivo

Permitir revisar la escena visual sin que el HUD bloquee la composición.

## Controles

- H: mostrar u ocultar HUD.
- M: alternar HUD completo o compacto.

## Razón

Durante pruebas visuales, screenshots y revisión de iluminación, el HUD completo puede tapar el Sol, planetas, lunas u órbitas.
