# Runtime HUD

## Objetivo

Mostrar información básica de la simulación directamente en pantalla.

## Información mostrada

- Nombre del motor.
- Fase actual.
- JD TDB.
- Días desde J2000.
- Escala temporal.
- Dirección del tiempo.
- Estado de pausa.
- Controles principales.

## Razón

El HUD permite depurar SimulationTime sin depender únicamente de la terminal.

## Estado

Implementado como SimulationHudPlugin.
